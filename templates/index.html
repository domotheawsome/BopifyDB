Bopify.


Bopify stores about 80 million songs and 400 million user profiles, this database allows users to upload music, create playlists, and search for music.


Many people find it convenient to have one place to find all of their favorite music. Our database driven website, Bopify, will allow users to search for music and create playlists. We are inspired to create this app based on Spotify, which currently stores about 80 million songs and 400 million user profiles. Specifically, the database driven website will record song information (artist name, song name, album name, genre) and user information (username, email, playlists).
